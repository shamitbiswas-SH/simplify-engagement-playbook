Connect your Google Sheet

This page has no sheet linked yet. Open index.html and set SHEET_ID at the top of the script to your Google Sheet ID (the long string in the sheet URL between /d/ and /edit).

  1. In Google Sheets: File > Share > Publish to web (the whole document, as web page).
  2. Also set sharing to "Anyone with the link: Viewer". Both are needed for the gviz feed.
  3. Keep the data tab named Playbook, or update SHEET_TAB in the script.
  4. Commit and push. The page reads the sheet live on every load.

Could not load the sheet

Check that the sheet is published to web, link sharing is on, and the tab name matches SHEET_TAB.
